智能知识库系统 - Windows打包说明
========================================

本说明介绍如何将项目打包成Windows应用程序，供用户一键安装使用。

📦 打包方案选择
---------------

方案一：PyInstaller打包（独立exe，推荐）
----------------------------------------
优点：
- 用户无需安装Python环境
- 一键运行，使用简单
- 所有依赖都打包在一起

缺点：
- 打包文件较大（约500MB-1GB）
- 首次打包时间较长（10-30分钟）

使用方法：
1. 运行 build_windows.bat
2. 等待打包完成
3. 打包文件位于 dist\ 目录
4. 将 dist\ 文件夹压缩后分发给用户

用户使用：
1. 解压文件
2. 双击"启动应用.bat"即可使用


方案二：安装包方案（需要Python环境）
------------------------------------
优点：
- 打包文件小
- 安装快速

缺点：
- 用户需要先安装Python 3.11+

使用方法：
1. 运行 build_simple.bat
2. 打包文件位于 dist\智能知识库系统-安装包\
3. 将文件夹压缩后分发给用户

用户使用：
1. 解压文件
2. 双击 install.bat 安装
3. 使用桌面快捷方式启动应用


🚀 快速开始
-----------

如果您是开发者，想打包项目：

【推荐】使用PyInstaller打包：
  build_windows.bat

创建安装包（用户需要Python）：
  build_simple.bat

如果您是用户，想安装使用：

有Python环境：
  1. 解压文件
  2. 运行 install.bat

没有Python环境：
  1. 使用PyInstaller打包版本
  2. 解压后直接运行"启动应用.bat"


📋 打包前准备
-------------

1. 确保Python 3.11+已安装
2. 安装所有依赖：
   pip install -r requirements.txt
3. 测试应用可以正常运行：
   streamlit run knowledge_base_deepseek.py


📁 文件说明
-----------

打包相关文件：
- build_windows.bat      - PyInstaller打包脚本
- build_windows.spec     - PyInstaller配置文件
- build_simple.bat       - 简化打包脚本
- install.bat            - 一键安装脚本
- launcher.py            - 启动器脚本

文档文件：
- 打包说明.txt          - 本文件
- 使用说明-打包版.txt   - 打包版本使用说明
- docs/WINDOWS_PACKAGING.md - 详细打包文档


⚠️ 注意事项
-----------

1. 打包前确保所有功能测试通过
2. 首次打包可能需要较长时间，请耐心等待
3. 打包后的文件较大是正常的（包含Python解释器）
4. 建议在干净的Windows系统上测试打包结果
5. 如果遇到问题，查看 docs/WINDOWS_PACKAGING.md


💡 提示
-------

- PyInstaller打包版本适合分发给没有Python环境的用户
- 安装包版本适合分发给有Python环境的用户
- 可以根据目标用户选择合适的方案
